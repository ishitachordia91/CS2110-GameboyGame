Game Description:
To win this game, you have to use the up, down, right, and left arrow keys to move Princess Peach around the screen. The objective is to reach Mario and Luigi without being killed. You have 5 lives. 

Pressing the delete/backspace key at any point during the game takes you to the start menu.
